# Mentel

**Category:** Full Stack (Frontend built with React)
**Role:** Frontend Developer (React)
**Year:** 2025

## About

Mentel is a comprehensive mental-health practice management platform that brings
the entire clinical operation into one place — patient and provider management,
intake, real-time messaging, payroll, analytics dashboards, and a full support &
ticketing system. The complete frontend was built in React, translating complex
admin workflows into a clean, data-dense, and intuitive interface used by
super-admins, providers, and staff.

## Key Features

- **Analytics Dashboard** — multi-tab view (Patient, Providers, Financial,
  Marketing, Support) with KPI cards, month-over-month trend indicators, and an
  interactive active-client-base & new-intake chart plus a key-metrics panel.
- **Real-time Messaging** — inbox/groups split, active vs. archived chats,
  availability/presence status, read receipts, search, and date-grouped threads.
- **Support & Ticketing** — sortable/filterable ticket table with Ticket ID,
  issue, requester, priority (Critical/High/Medium), complexity (Standard,
  Advanced, Multi-Step, Project Level), assignment, status, SLA-breach tracking,
  and pagination (120 tickets across pages).
- **Secure Sign-In** — branded, enterprise-compliant login experience.
- **Core Modules** — Patient Management, Provider Management, Intake, Payroll,
  User Management, and Settings, all under a collapsible sidebar.

## Tech Stack

React · Redux · Chart.js · REST APIs · Tailwind CSS
